Política de Privacidade

A Bug Busters Engenharia valoriza a privacidade e segurança dos dados de clientes e potenciais clientes. Esta política descreve como coletamos, usamos e protegemos as informações pessoais, conforme a Lei Geral de Proteção de Dados Pessoais (LGPD).

Informações que Coletamos

Coletamos as seguintes informações pessoais fornecidas voluntariamente pelos usuários ao preencher nossos formulários de contato:

  • Nome: identificação do usuário;
  • E-mail: endereço de contato;
  • Telefone e WhatsApp: números para comunicação;
  • Área de atuação da empresa: setor de atividade do usuário;
  • Interesses: serviços e soluções que o usuário deseja conhecer.

Como Coletamos os Dados

Os dados são coletados diretamente quando você preenche nossos formulários de contato. Utilizamos o Zoho Mail para armazenar e proteger essas informações com segurança.

Finalidade do Uso dos Dados

As informações são usadas para geração de leads e estabelecimento de contato comercial, visando apresentar soluções personalizadas em desenvolvimento e consultoria de software em nuvem.

Compartilhamento de Dados

Seus dados não são compartilhados com terceiros para fins comerciais. Utilizamos as informações apenas internamente, exceto quando exigido por lei.

Direitos dos Titulares

Você tem o direito de:

  • Confirmar o tratamento dos seus dados;
  • Corrigir informações incorretas;
  • Solicitar exclusão dos dados;
  • Portar seus dados para outro fornecedor;
  • Revogar consentimentos dados anteriormente.

Para exercer esses direitos, entre em contato pelo e-mail informado abaixo.

Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados contra acessos não autorizados e outras ameaças.

Contato

Para dúvidas ou solicitações relacionadas a seus dados, envie um e-mail para contato@bugbusters.eng.br. Responderemos dentro dos prazos legais.

Atualizações desta Política

Esta política poderá ser atualizada para refletir alterações nas práticas ou legislação. Consulte esta página regularmente.
